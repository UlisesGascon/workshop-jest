# workshop-jest
Jest Workshop

## Content

### Installation and scripts

Installation

```
npm i -D jest
```

Npm Scripts

```
"test": "jest  --verbose ./**/*.test.js",
"test:coverage": "jest --verbose ./**/*.test.js --coverage",
"test:watch": "jest --verbose ./**/*.test.js --watchAll",
```
